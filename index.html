Theme

Building Tomorrow's Solutions Today

Hackegerton 2026 invites you to innovate, create, and transform ideas into reality. Our grand theme focuses on empowering students to build solutions that address real-world challenges through technology. Whether you're passionate about artificial intelligence, web development, blockchain, or any other tech domain, this hackathon is your platform to showcase creativity, collaborate with peers, and make a meaningful impact. Join us in shaping the future, one line of code at a time.

Tracks

AI & Data Track

Dive deep into the world of artificial intelligence and data science. This track focuses on building intelligent systems that can learn, adapt, and make data-driven decisions. Whether you're working with machine learning models, neural networks, or data analytics, this track empowers you to create solutions that understand patterns, predict outcomes, and automate complex processes.

Sample Projects You Can Build

  • Predictive Analytics Platform: Build a system that predicts trends, customer behavior, or market movements using historical data
  • AI Chatbot Assistant: Create an intelligent chatbot that can answer questions, provide support, or assist users in various tasks
  • Image Recognition System: Develop a model that can identify objects, faces, or patterns in images
  • Recommendation Engine: Build a system that suggests products, content, or services based on user preferences
  • Data Visualization Dashboard: Create interactive dashboards that transform complex data into meaningful insights

Web / Mobile Track

Create beautiful, functional, and user-friendly applications for web and mobile platforms. This track covers everything from responsive web design to native mobile apps, focusing on creating seamless user experiences across all devices. Whether you're building a progressive web app, a React Native application, or a full-stack web solution, this track helps you bring your ideas to life on screens of all sizes.

Sample Projects You Can Build

  • E-Commerce Platform: Build a full-featured online store with shopping cart, payment integration, and order management
  • Social Media App: Create a mobile or web app for connecting users, sharing content, and building communities
  • Task Management Tool: Develop a productivity app for organizing tasks, projects, and team collaboration
  • Fitness Tracking App: Build an app that tracks workouts, nutrition, and health metrics with beautiful visualizations
  • Real-Time Chat Application: Create a messaging platform with instant communication, file sharing, and notifications

FinTech / Blockchain Track

Explore the future of finance and decentralized technology. This track combines financial innovation with blockchain technology, enabling you to build secure, transparent, and efficient financial solutions. From payment systems and digital wallets to smart contracts and DeFi platforms, this track empowers you to revolutionize how money moves and how financial services are delivered.

Sample Projects You Can Build

  • Digital Wallet Application: Build a secure mobile wallet for storing, sending, and receiving digital currencies
  • Smart Contract Platform: Create automated contracts that execute transactions when conditions are met, without intermediaries
  • Peer-to-Peer Payment System: Develop a platform for direct money transfers between users with low fees
  • NFT Marketplace: Build a platform for creating, buying, and selling non-fungible tokens
  • Cryptocurrency Exchange: Create a trading platform for buying, selling, and exchanging digital assets

Frequently Asked Questions
